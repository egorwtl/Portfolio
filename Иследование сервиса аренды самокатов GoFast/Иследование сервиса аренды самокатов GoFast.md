## Описание проекта
Проведен предварительный анализ использования тарифов на выборке клиентов сервиса самокатов, проанализировано поведение клиентов при использовании услуг. Проведена предобработка данных, их анализ. Проверены статистические гипотезы на основе имеющихся данных.

## Навыки и инструменты
`Python` `Pandas` `NumPy` `Matplotlib` `SciPy` `описательная статистика` `проверка статистических гипотез`

## Общий вывод
Пользователи с подпиской приносят больше прибыли для сервиса GoFast, поэтому задача по увеличению количества пользователей с подпиской, поставленная продакт-менеджерами, вполне обоснованная и релевантная.
